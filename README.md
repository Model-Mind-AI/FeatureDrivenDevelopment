# FeatureDrivenDevelopment

Repository scaffold for building **Cursor skills** and **toolkit plugins** (MCP servers, CLIs, hook helpers) to accelerate software development.

## Repository layout

- **`.cursor/skills/`** — Project-scoped skills (`<name>/SKILL.md`). Start from `_template`.
- **`toolkit/plugins/`** — Source for plugins and reusable tooling that are not Cursor skills.
- **`toolkit/mcp/`** — MCP server projects or wrappers for use with Cursor.
- **`AGENTS.md`** — Quick reference for humans and agents working in this repo.

See `AGENTS.md` for conventions and how to add new skills or plugins.
